Q0 · Ecosistema

¿Cómo se relacionan entre sí las cuatro grandes áreas del campo neuromusical: transducción coclear, entrainment, conectividad fMRI y clínica?

Ver respuesta

Forman una cascada causal: la transducción coclear (Gammatone, ERB, IHC phase-locking) convierte la señal acústica en actividad neural subcortical. El entrainment (PLV, FFR, Lenguas de Arnold, Kuramoto) describe cómo las oscilaciones neurales se sincronizan con el ritmo musical. La conectividad fMRI (dFC, grafos, VBM) revela cómo estas sincronizaciones reorganizan redes cerebrales a gran escala y producen neuroplasticidad estructural. La clínica (MIT, RAS, Brain.fm, taVNS) explota cada uno de estos mecanismos para diseñar intervenciones terapéuticas basadas en evidencia.

M1 · Transducción Coclear
Tonotopía y Banco Gammatone
La cóclea actúa como una FFT biológica: la membrana basilar descompone frecuencias de forma tonotópica, con células ciliadas externas aplicando compresión no lineal.
Teoría Lab

Membrana Basilar y ERB

Tonotopía y Filtro Gammatone

La rigidez decrece desde la base (20 kHz) hasta el ápice (20 Hz). Las OHC aplican ganancia no lineal compresiva de ~40 dB. El filtro gammatone modela la respuesta de la membrana basilar:

gt(t) = t^(n-1) · cos(2πfc·t + φ) · exp(-2π·b·t)
ERB(f) = 24.7 · (4.37·f/1000 + 1)

El phase-locking de las IHC permite codificar la microestructura temporal hasta ~5 kHz en mamíferos — límite crítico para el reconocimiento de fonemas.

Banco Gammatone en Python

import numpy as np
from scipy.signal import fftconvolve

def erb(fc):
    return 24.7 * (4.37 * fc / 1000 + 1)

def gammatone_filter(fc, fs, n=4, duration=0.05):
    t = np.arange(0, duration, 1/fs)
    b = 1.019 * erb(fc)
    envelope = t**(n-1) * np.exp(-2 * np.pi * b * t)
    carrier = np.cos(2 * np.pi * fc * t)
    ir = envelope * carrier
    return ir / np.sum(np.abs(ir))

def gammatone_filterbank(signal, fs, num_channels=20, f_low=250, f_high=8000):
    fc_array = np.linspace(f_low, f_high, num_channels)
    output = np.zeros((num_channels, len(signal)))
    for i, fc in enumerate(fc_array):
        ir = gammatone_filter(fc, fs)
        output[i] = fftconvolve(signal, ir, mode='same')
    return output, fc_array

Q1 · Gammatone vs FFT

¿Por qué un banco de filtros gammatone es más apropiado que una FFT estándar para modelar la cóclea?

Ver respuesta

La FFT tiene resolución frecuencial uniforme (Δf constante), pero la cóclea tiene resolución logarítmica (ERB proporcional a fc): distingue mejor frecuencias bajas (alta resolución espectral) y peor altas. El filtro gammatone captura esto con ancho de banda ERB proporcional a fc. Además, su respuesta temporal (envolvente exponencial × portadora cosenoidal) modela fielmente la respuesta transitoria de las fibras nerviosas auditivas, que la FFT ignora por completo — crítico para el análisis de phase-locking y la microestructura temporal.

M2 · Phase-Locking y FFR
Frequency-Following Response y Phase-Locking Value
La FFR refleja el bloqueo de fase de poblaciones neuronales del tronco encefálico auditivo en respuesta a la microestructura periódica del estímulo.
Teoría Lab

FFR y PLV

PLV = |1/N · Σ exp(j·Δφ_n)|
FFR(t) = (1/N) Σ EEG_n(t − t_onset)

PLV = 1 indica sincronización perfecta; PLV = 0 indica distribución de fase uniforme. La FFR refleja actividad del Núcleo Coclear, Complejo Olivar Superior y Colículo Inferior. Se extrae del electrodo Cz (referenciado a mastoides) promediando coherentemente 1000-6000 ensayos.

import numpy as np
from scipy.signal import hilbert, butter, filtfilt

def compute_plv(epochs, stim_freq, sfreq):
    N, T = epochs.shape
    t = np.arange(T) / sfreq
    stim_phase = 2 * np.pi * stim_freq * t
    b, a = butter(4, [stim_freq*0.8, stim_freq*1.2], btype='band', fs=sfreq)
    plv_values = []
    for trial in epochs:
        filtered = filtfilt(b, a, trial)
        eeg_phase = np.angle(hilbert(filtered))
        phase_diff = eeg_phase - stim_phase
        plv_values.append(np.abs(np.mean(np.exp(1j * phase_diff))))
    return np.mean(plv_values)

def extract_ffr(epochs):
    return np.mean(epochs, axis=0)

Q2 · FFR y promediado

¿Por qué la FFR requiere miles de ensayos para ser visible en el EEG?

Ver respuesta

La FFR tiene amplitud de ~0.1 µV versus ruido EEG espontáneo de ~20 µV. Al promediar N ensayos, la señal coherente (bloqueada en fase con el estímulo) crece linealmente (N·A) mientras el ruido aleatorio crece solo como √N, dando SNR ∝ √N. Con N=3000 ensayos, la ganancia de SNR es ~55× respecto a un solo ensayo. Sin este promediado coherente, la FFR quedaría completamente sepultada bajo el EEG espontáneo.

M3 · Lenguas de Arnold
Dinámica de Fase y Zonas de Enganche Neuronal
Las Lenguas de Arnold modelan las regiones de parámetros donde dos osciladores acoplados se sincronizan, con aplicación directa al entrainment cerebro–estímulo musical.
Teoría Lab

Mapa de Círculo y Modelo de Kuramoto

t_n = t_{n-1} + c/a + (b/a)·sin(2π·t_{n-1})
dφ_i/dt = ω_i + (K/N)·Σ sin(φ_j − φ_i)

El parámetro de orden r = |Σ exp(j·φ_i)|/N mide la coherencia colectiva. Las Lenguas de Arnold son regiones en el espacio (detuning, coupling) donde la ratio de bloqueaje es racional p:q. A mayor acoplamiento K, mayor amplitud de la lengua (mayor tolerancia al detuning).

import numpy as np

def winding_number(omega, K, n_iter=1000, n_transient=200):
    theta = 0.0
    thetas = []
    for i in range(n_iter + n_transient):
        theta_new = theta + omega - (K/(2*np.pi)) * np.sin(2*np.pi*theta)
        if i >= n_transient:
            thetas.append(theta_new - theta)
        theta = theta_new
    return np.mean(thetas)

# Mapa de Arnold Tongues
omega_range = np.linspace(0, 1, 300)
K_range = np.linspace(0, 1, 200)
WN = np.zeros((len(K_range), len(omega_range)))
for i, K in enumerate(K_range):
    for j, omega in enumerate(omega_range):
        WN[i, j] = winding_number(omega, K)

Q3 · Lenguas de Arnold

¿Por qué las Lenguas de Arnold tienen forma de cuña que se ensancha a mayor acoplamiento?

Ver respuesta

Imagina dos relojes con frecuencias ligeramente diferentes conectados con un resorte. Con resorte débil, solo se sincronizan si sus frecuencias ya son casi iguales (lengua estrecha). Con resorte fuerte, pueden sincronizarse aunque sus frecuencias difieran considerablemente (lengua ancha). En el cerebro, la intensidad del estímulo acústico actúa como esa "fuerza de acoplamiento": estímulos más intensos generan zonas de entrainment más amplias (toleran mayor detuning entre la frecuencia neural y la del estímulo). Esto explica por qué el protocolo RAS comienza al 10% sobre la cadencia espontánea: hay que estar dentro de la lengua para lograr acoplamiento 1:1.

M4 · Entrainment y Entropía
Entropía de Shannon Normalizada y Modelo DISE
La entropía de la distribución de diferencias de fase entre EEG y estímulo cuantifica el grado de entrainment. ρ̃ = 1 indica acoplamiento perfecto.
Teoría Lab

Entropía de Shannon y ρ̃

S = −Σ p_k · ln(p_k) (entropía de Shannon de la distribución Δφ)
ρ̃ = (S_max − S) / S_max → ρ̃ ∈ [0, 1]

ρ̃ = 1: distribución de Dirac (entrainment perfecto). ρ̃ = 0: distribución uniforme (sin acoplamiento). El modelo DISE (Dynamic Information Selection by Entrainment) postula que el phase-resetting cortical al ritmo del estímulo crea ventanas temporales de alta excitabilidad que priorizan dinámicamente el procesamiento de información relevante.

import numpy as np
from scipy.signal import hilbert, butter, filtfilt

def normalized_entropy(phase_diff, n_bins=18):
    phase_diff_norm = phase_diff % (2 * np.pi)
    counts, _ = np.histogram(phase_diff_norm, bins=n_bins, range=(0, 2*np.pi))
    probs = counts / counts.sum()
    probs = probs[probs > 0]
    S = -np.sum(probs * np.log(probs))
    S_max = np.log(n_bins)
    return (S_max - S) / S_max

def compute_entrainment(eeg, stim_freq, sfreq):
    b, a = butter(4, [stim_freq*0.5, stim_freq*2.0], btype='band', fs=sfreq)
    eeg_filtered = filtfilt(b, a, eeg)
    eeg_phase = np.angle(hilbert(eeg_filtered))
    t = np.arange(len(eeg)) / sfreq
    stim_phase = (2 * np.pi * stim_freq * t) % (2 * np.pi)
    return normalized_entropy(eeg_phase - stim_phase)

Q4 · Brain.fm AM vs binaural

¿Por qué Brain.fm usa modulación de amplitud en lugar de binaural beats para inducir entrainment theta?

Ver respuesta

La modulación de amplitud directa (portadora de 200 Hz modulada a 4 Hz) produce una señal con envolvente temporal de 4 Hz que la corteza auditiva primaria procesa de forma robusta y directa, generando phase-locking medible en EEG. Los binaural beats (ej. 200 Hz oído izq. + 204 Hz oído der. = 4 Hz percibido) son una ilusión auditiva: el cerebro percibe la diferencia pero con gran variabilidad entre personas y evidencia de entrainment débil. El modelo DISE predice que la AM directa propagará ventanas de alta excitabilidad a redes de atención y memoria de trabajo de forma más robusta que los binaural beats.

M5 · Algoritmo CIS
Continuous Interleaved Sampling en Implantes Cocleares
CIS es la arquitectura DSP estándar de los implantes cocleares modernos: banco de filtros → extracción de envolvente → mapeo dinámico → pulsos bifásicos entrelazados.
Teoría Lab

Pipeline CIS

5 etapas del algoritmo CIS

  1. Pre-énfasis: Filtro pasa-altas con rolloff de -6 dB/oct bajo 1200 Hz
  2. Banco de filtros Butterworth 6to orden: L canales logarítmicamente espaciados (6-22)
  3. Extracción de envolvente: Rectificación + LPF Butterworth 2do orden (200-400 Hz)
  4. Mapeo dinámico: Compresión logarítmica mapeando a THR–MCL del paciente
  5. Estimulación pulsátil: Pulsos bifásicos balanceados estrictamente entrelazados
Y = A·x^p + B, A = (MCL − THR)/(x_max^p − x_min^p), B = THR − A·x_min^p

Q5 · Pulsos entrelazados CIS

¿Qué problema clínico específico resuelve el entrelazamiento de pulsos en CIS?

Ver respuesta

La perilinfa es un fluido conductor que rodea los electrodos. Si dos electrodos próximos (separados ~2 mm) disparan simultáneamente, sus campos eléctricos se superponen y pueden estimular fibras nerviosas de las frecuencias "incorrectas" (diafonía). Al entrelazar los pulsos temporalmente — cada canal dispara en un slot diferente dentro de la ventana de 4 ms — cada electrodo está activo solo brevemente y su campo colapsa antes de que el siguiente dispare, preservando la selectividad tonotópica del implante. Es la razón por la que el CIS puede mantener 6-22 canales funcionalmente independientes.

M6 · Simulación IC
Vocoder de Implante Coclear y Ecuación de Greenwood
La simulación del implante permite evaluar inteligibilidad de voz, calcular errores de inserción de electrodos y optimizar el mapeo de canales usando la ecuación de Greenwood.
Teoría Lab

Greenwood y Error de Inserción

f(x) = A · (10^(a·x) − k) donde A=165.4, a=0.06, k=0.88
S(t) = Σ_i A_i(t) · sin(2π·F_i·t) (resíntesis vocoder IC)

Si un arreglo de electrodos se inserta solo hasta 22 mm en vez de 25 mm, los electrodos apicales mapean frecuencias más altas de lo óptimo (frequency upshift), causando errores de discriminación vocálica.

import numpy as np
from scipy.signal import butter, sosfiltfilt

def greenwood(x_mm):
    A, a, k = 165.4, 0.06, 0.88
    return A * (10**(a * x_mm) - k)

def ci_vocoder_resynth(signal, fs, n_ch=8, insertion_mm=25.0):
    electrode_pos = np.linspace(2, insertion_mm, n_ch)
    center_freqs = np.array([greenwood(x) for x in electrode_pos])
    center_freqs = np.clip(center_freqs, 100, fs/2 - 100)
    output = np.zeros(len(signal))
    for i, fc in enumerate(center_freqs):
        bw = fc * 0.4
        fl = max(50, fc - bw/2)
        fh = min(fs/2 - 100, fc + bw/2)
        sos = butter(3, [fl, fh], btype='band', fs=fs, output='sos')
        band = sosfiltfilt(sos, signal)
        envelope = np.abs(band)
        sos_lp = butter(2, 400, btype='low', fs=fs, output='sos')
        envelope = sosfiltfilt(sos_lp, envelope)
        t = np.arange(len(signal)) / fs
        output += envelope * np.sin(2 * np.pi * fc * t)
    return output / (np.max(np.abs(output)) + 1e-8)

Q6 · Vocoder y envolvente

¿Por qué el vocoder de IC usa envolventes lentas (LPF a 400 Hz) en lugar de la señal de banda completa para la resíntesis?

Ver respuesta

Los electrodos del IC estimulan con pulsos eléctricos a tasas de ~1000-5000 pulsos/seg por canal. La microestructura fina de la señal acústica (variaciones a 8000 Hz) no puede ser transmitida fielmente a estas tasas. Lo que sí porta información inteligible es la envolvente (hasta ~400 Hz): ritmo, prosodia, formantes vocálicos, dinámica fonética. Por eso el IC extrae solo la envolvente lenta de cada canal y la usa para modular la amplitud de los pulsos eléctricos, descartando la microestructura temporal fina. Esto explica por qué los usuarios de IC comprenden voz pero tienen dificultades con la percepción de pitch musical.

Q7 · Movimiento en fMRI musical

¿Por qué el movimiento de cabeza es un confound tan crítico en estudios fMRI de música?

Ver respuesta

En experimentos de neuroimagen musical, los sujetos tienden a moverse sutilmente al ritmo de la música (micro-movimientos rítmicos). Estos movimientos periódicos generan cambios en la señal BOLD que se correlacionan temporalmente con el paradigma musical, creando "activaciones" espurias. El FD > 0.3 mm como criterio de scrubbing elimina los volúmenes rítmicamente contaminados. Es por eso que los estudios de neuroimagen musical más rigurosos exigen controles de movimiento más estrictos que estudios de cognición estática: la señal de interés (respuesta neural al ritmo) puede ser indistinguible del artefacto de movimiento rítmico sin un QC cuidadoso.

M8 · dFC Dinámica
Conectividad Funcional Dinámica con Sliding-Window
La dFC captura cómo las redes cerebrales reorganizan su conectividad a lo largo del tiempo durante la escucha musical, revelando "estados" cerebrales discretos mediante clustering.
Teoría Lab

Sliding Window y K-Means

FC(t_k) = corrcoef(X[t_k : t_k+W]) ∀ ventana k (W = 60s, step = 1 TR)
min Σ_k Σ_i ||FC_k − μ_c||² (K-Means, K=2..8 estados)
import numpy as np
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score

def compute_dfc(time_series, window_size=30, step=1):
    n_tp, n_roi = time_series.shape
    windows = []
    for start in range(0, n_tp - window_size, step):
        segment = time_series[start:start + window_size]
        fc = np.corrcoef(segment.T)
        windows.append(fc)
    return np.array(windows)  # (n_windows, n_roi, n_roi)

def cluster_brain_states(dfc_matrices, k_range=(2, 8)):
    n_win, n_roi, _ = dfc_matrices.shape
    idx = np.triu_indices(n_roi, k=1)
    X = np.arctanh(np.clip(dfc_matrices[:, idx[0], idx[1]], -0.99, 0.99))
    best_k, best_score, best_labels = None, -1, None
    for k in range(k_range[0], k_range[1] + 1):
        km = KMeans(n_clusters=k, random_state=42, n_init=10)
        labels = km.fit_predict(X)
        score = silhouette_score(X, labels)
        if score > best_score:
            best_score = score; best_k = k; best_labels = labels
    return best_labels, best_k

Q8 · Limitación sliding-window

¿Cuál es la limitación principal del método sliding-window para calcular dFC?

Ver respuesta

El tamaño de ventana introduce un trade-off fundamental: ventanas cortas (15 TRs = 30s) producen estimaciones de correlación inestables con pocas muestras (alta varianza, poco fiables); ventanas largas (60 TRs = 120s) tienen baja resolución temporal y pueden mezclar múltiples estados cerebrales dentro de una sola ventana, promediando y ocultando las transiciones. Con TR=2s y ventana de 30 TRs (60s), el equilibrio es razonable pero no perfecto. Métodos alternativos como HMM (Hidden Markov Model) o co-activation patterns evitan este problema pero tienen sus propias asunciones y limitaciones.

M9 · Grafos Neuronales
Teoría de Grafos para Redes Cerebrales
Las métricas de graph theory (Eficiencia Global, Modularidad, Centralidad) cuantifican cómo el cerebro integra y segrega información durante la experiencia musical.
Teoría Lab

Métricas Clave de Network Science

E_glob = 1/(N(N-1)) · Σ_{i≠j} 1/d(i,j)
Q = Σ_c [L_c/L − (d_c/2L)²] (Modularidad Louvain)
C_i = 2·t_i / (k_i(k_i−1)) donde t_i = triángulos en vecindad de i
import numpy as np
import networkx as nx

def fc_to_graph(fc_matrix, threshold=0.2):
    n = fc_matrix.shape[0]
    fc_abs = np.abs(fc_matrix.copy())
    np.fill_diagonal(fc_abs, 0)
    vals = fc_abs[np.triu_indices(n, k=1)]
    cutoff = np.percentile(vals, (1 - threshold) * 100)
    adj = (fc_abs >= cutoff).astype(int)
    return nx.from_numpy_array(adj)

def compute_graph_metrics(G):
    if not nx.is_connected(G):
        G = G.subgraph(max(nx.connected_components(G), key=len)).copy()
    return {
        'E_glob': nx.global_efficiency(G),
        'E_loc': nx.local_efficiency(G),
        'clustering': nx.average_clustering(G),
        'L': nx.average_shortest_path_length(G)
    }

Q9 · Webern vs Bach grafos

¿Por qué la música de Webern activa mayor Eficiencia Global en cerebros expertos que Bach?

Ver respuesta

Bach tiene alta predictibilidad métrica: el cerebro activa módulos locales (auditivo, motor) que se sincronizan con el ritmo sin necesidad de comunicación global. Webern viola constantemente las expectativas tonales y rítmicas: el Error de Predicción (Predictive Coding) activa el córtex cingulado anterior, pars opercularis y regiones frontales que deben comunicarse entre sí para resolver la ambigüedad. Esta comunicación global distribuida se refleja en alta Eficiencia Global en el grafo cerebral. El cerebro experto (con vocabulario musical aprendido) puede hacer este procesamiento; el no músico simplemente se "desconecta" ante Webern, sin el beneficio del procesamiento global.

Q10 · Neuroplasticidad musical

¿Por qué el entrenamiento musical produce cambios en VBM detectables (neuroplasticidad estructural) y no solo funcionales?

Ver respuesta

La práctica instrumental intensiva (>10 años, 30+ min/día) genera cambios físicos en el tejido cerebral: más dendritas y sinapsis (sinaptogénesis), mayor mielinización de axones que conectan regiones auditivo-motoras, y más vasos sanguíneos (angiogénesis) para suministrar más oxígeno. Todos estos cambios aumentan la densidad local de materia gris en regiones como la corteza auditiva primaria, SMA y cerebelo. VBM detecta estos aumentos comparando densidad de GM vóxel a vóxel con un GLM. El fascículo arqueado (DTI/FA) refleja la mielinización mejorada de la vía auditivo-motora. Es el sustrato estructural de la habilidad motora musical adquirida.

M11 · MIT y Neurorrehabilitación

Mecanismo de compensación hemisférica

Q11 · MIT y hemisferio derecho

¿Por qué la prosodia exagerada (entonación fuerte) de la MIT activa el hemisferio derecho más que el habla normal?

Ver respuesta

El hemisferio derecho es dominante para procesar contorno melódico, ritmo y prosodia, mientras el izquierdo es dominante para sintaxis propositiva y vocabulario. La MIT convierte las palabras en fragmentos de melodía (Sprechgesang), reclutando el hemisferio derecho (intacto en la afasia de Broca). El tapping con la mano izquierda (controlada por el hemisferio derecho) refuerza la predicción temporal, y con práctica repetida el hemisferio derecho desarrolla una vía alternativa para el lenguaje propositivo mediada por el fascículo arqueado derecho. Estudios DTI muestran que este fascículo crece estructuralmente tras MIT exitosa, confirmando la neuroplasticidad inducida.

Q12 · RAS y Lenguas de Arnold

¿Por qué el RAS debe comenzar al 10% sobre la cadencia espontánea del paciente, y no directamente al objetivo terapéutico?

Ver respuesta

El sistema motor debe "capturarse" (entrainment) al ritmo primero, y solo puede hacerlo si el ritmo externo está dentro de la Lengua de Arnold del oscilador motor del paciente. Un salto grande al objetivo (ej. de 80 BPM a 110 BPM de golpe = 37.5% de diferencia) excede la zona de captura y el entrainment 1:1 no ocurre. Al comenzar al 10% sobre la cadencia espontánea (88 BPM), el metrónomo está dentro de la zona de sincronización 1:1, y el paciente naturalmente ajusta su paso. Semana a semana, se incrementa el BPM ±5%, arrastrando gradualmente el oscilador motor hacia la cadencia objetivo sin romper el acoplamiento.

import numpy as np
from scipy.io import wavfile

def generate_entrainment_audio(duration_s=600, fs=44100,
                               carrier_hz=200, mod_hz=4.0, mod_index=0.8):
    t = np.linspace(0, duration_s, int(duration_s * fs))
    carrier = np.sin(2 * np.pi * carrier_hz * t)
    carrier += 0.5 * np.sin(2 * np.pi * carrier_hz * 2 * t)
    carrier /= 1.5
    mod_env = 1.0 + mod_index * np.sin(2 * np.pi * mod_hz * t)
    fade = int(10 * fs)
    mod_env[:fade] *= np.linspace(0, 1, fade)
    mod_env[-fade:] *= np.linspace(1, 0, fade)
    output = carrier * mod_env
    return (output / np.max(np.abs(output)) * 32767).astype(np.int16)

Q13 · taVNS y entrainment

¿Por qué el taVNS potencia el efecto del entrainment auditivo en aplicaciones terapéuticas?

Ver respuesta

El entrainment auditivo es más efectivo cuando la corteza está en un estado de alta plasticidad y receptividad. El taVNS activa el nervio vago (CN X) que proyecta al Locus Coeruleus (LC), el principal "grifo" de noradrenalina del cerebro. La noradrenalina aumenta la señal-to-ruido neuronal: las neuronas corticales responden de forma más nítida a estímulos relevantes (mayor "gain"). En este estado de alta receptividad, el entrainment auditivo (AM-theta) puede establecer phase-locking más robusto. La combinación de neuromodulación bottom-up (taVNS) con entrainment top-down (ritmo auditivo) produce efectos sinérgicos en aplicaciones de atención, recuperación de afasia y neurorrehabilitación.

E1 · IC + FFR + Entrainment

Un paciente con IC tiene 8 canales, inserción de 22mm. ¿Qué problema de entrainment predices para la FFR?

Ver respuesta

El frequency upshift desplaza los canales apicales hacia frecuencias más altas: las fibras nerviosas activas no corresponden al tonotipo correcto para las frecuencias de baja frecuencia. Por ejemplo, el canal que debería procesar 250 Hz en una inserción de 25 mm ahora estimula fibras de ~500 Hz (inserción de 22 mm según Greenwood). El PLV para frecuencias graves será reducido porque las fibras no están "sintonizadas" para esa frecuencia — el phase-locking es subóptimo. La FFR mostrará latencias alteradas y menor amplitud espectral en el fundamental, con armónicos distorsionados. El ρ̃ del entrainment en banda baja sera anómalamente bajo en estos pacientes.

E2 · RAS y Lenguas de Arnold

¿Por qué la RAS a 110 BPM podría NO funcionar para un paciente Parkinson con cadencia espontánea de 80 BPM?

Ver respuesta

110/80 = 1.375: este ratio excede la anchura de la Lengua de Arnold 1:1 para el oscilador motor del paciente. El cerebelo no puede acoplarse (entrainment) a un ritmo tan distante de su frecuencia natural, y el acoplamiento 1:1 no ocurre. El protocolo RAS comienza al 10% sobre la cadencia espontánea (88 BPM) porque este detuning está dentro de la zona de captura 1:1. Semana a semana se incrementa el BPM ±5%, arrastrando gradualmente el oscilador motor hacia 110 BPM sin salir de la zona de entrainment. Es la topología de las Lenguas de Arnold aplicada directamente a la neurorrehabilitación motora.

E3 · VBM y covariables

¿Por qué un estudio VBM de neuroplasticidad musical necesita controlar el IQ como covariable?

Ver respuesta

El IQ correlaciona con mayor volumen de GM en áreas prefrontales que coinciden con regiones afectadas por el entrenamiento musical. Si los grupos músico vs control difieren accidentalmente en IQ medio (posiblemente porque el acceso a educación musical enriquecida correlaciona con entornos socioeconómicos de mayor estimulación cognitiva), las diferencias de GM observadas podrían reflejar efectos del IQ y no del entrenamiento musical per se. Incluir IQ como covariable en el GLM "elimina" este efecto compartido, dejando solo la varianza explicada por el entrenamiento. Es el mismo principio que controlar edad, sexo y lateralidad para aislar el efecto de interés en cualquier estudio de neuroimagen.

E4 · CIS y percepción musical

¿Por qué los usuarios de IC comprenden voz en silencio pero tienen dificultades para reconocer música?

Ver respuesta

El IC codifica la envolvente lenta (hasta 400 Hz) que porta información fonética suficiente para voz: los formantes vocálicos (250-3500 Hz) y su dinámica temporal son los portadores principales de la inteligibilidad del lenguaje. La percepción musical, en cambio, requiere: pitch exacto (microestructura temporal fina que el CIS descarta con el LPF de envolvente), diferencias de timbre (espectro de armónicos que el vocoder distorsiona con solo 8-22 bandas), y relaciones armónicas entre tonos simultáneos (polifonía). Con la envolvente de 400 Hz como única información temporal, la representación musical queda brutalmente empobrecida — suficiente para el ritmo pero no para la melodía, armonía ni timbre.

E5 · Síntesis trimodal longitudinal

¿Cómo integrarías FFR, dFC y VBM en un único estudio longitudinal de plasticidad musical?

Ver respuesta

Diseño trimodal: FFR (marcador subcortical de phase-locking, ms) + dFC (reorganización funcional de redes, minutos-meses) + VBM (cambios estructurales, años) medidos pre/post 6 meses de entrenamiento. La hipótesis es una cascada temporal: FFR mejora primero (semanas, sincronización subcortical más eficiente), luego dFC se reorganiza (meses, redes auditivo-motoras más eficientes), finalmente VBM detecta consolidación estructural (6+ meses, mayor densidad de GM en corteza auditiva y SMA). Esta cascada permitiría identificar biomarcadores tempranos de plasticidad musical antes de que los cambios estructurales sean detectables, y validar el mecanismo causal de la neuroplasticidad musical a diferentes escalas temporales.
